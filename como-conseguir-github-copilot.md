# Como conseguir GitHub Copilot gratis siendo alumno de UTN


Estás justo en la página correcta. Hagámoslo **en 2 minutos**.

https://github.com/education

## 1️⃣ Click en el botón verde

En la pantalla que tenés abierta, hacé click en:

**“Join GitHub Education”**

Eso te lleva a la página de beneficios de estudiante.

---

## 2️⃣ Elegí **Student**

Te va a preguntar qué sos:

-   Student ✅
    
-   Teacher
    
-   School
    

Elegí **Student**.

---

## 3️⃣ Verificación de estudiante

Te va a pedir **una de estas dos cosas**:

### Método rápido (si tenés mail UTN)

Ponés tu mail institucional, por ejemplo:

```
@frt.utn.edu.ar  
@frba.utn.edu.ar  
@utn.edu.ar
```

GitHub lo verifica automáticamente.

---

### Método universal (funciona siempre)

Subís una foto nitida o un PDF de tu **constancia de alumno regular** o **constancia de inscripción**.:


Debe verse:

-   tu **nombre**

-   **UTN**

-   **fecha reciente**

---

## 4️⃣ Esperar aprobación

Normalmente tarda:

-   **5 minutos – 1 hora** si usa mail institucional
    
-   **1–3 días** si subís documento
    

---

## 5️⃣ Cuando te aprueban

Se activa el **GitHub Student Developer Pack** y obtenés gratis:

-   **GitHub Copilot**
    
-   **GitHub Codespaces**
    
-   dominios gratis
    
-   hosting
    
-   herramientas cloud
    

---

💡 **Tip importante (muy pocos lo saben):**  
Si ya tenés cuenta de **GitHub**, podés ir directo a:

```
https://github.com/settings/education/benefits
```

y aplicar ahí mismo.
