### Como configurar el entorno de desarrollo:

1. Se debe instalar `Visual Studio Code` (VSCode) o cualquier editor de texto de su preferencia.
> https://code.visualstudio.com

2. Se debe instalar el `SDK de .NET 10` desde la página oficial de Microsoft.
> https://dotnet.microsoft.com/es-es/download

3. Se debe instalar la extensión de `C# Dev Kit` para VSCode.
> https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csdevkit

4. Se debe instalar `Git for Windows`
> https://git-scm.com/install/windows

5. Se debe crear una cuenta de Github si no se tiene una.
> https://github.com/

6. Se debe instalar `Github Copilot` para VSCode (opcional, pero recomendado).

> Version gratuita limitada
https://github.com/settings/copilot

> Version gratuita para estudiantes
https://github.com/settings/education/benefits 

7. Se debe instalar `Github Desktop`
> https://desktop.github.com/

8. Se debe clonar el repositorio del curso utilizando Github Desktop o Git Bash.
> https://github.com/AlejandroDiBattista/tup26-p3#
